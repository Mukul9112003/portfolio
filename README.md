## Portfolio Projects


### 1. Salary Intelligence Platform

The Salary Intelligence Platform is an end-to-end machine learning system designed to analyze salary datasets and predict salary ranges based on different features such as job role, experience level, and industry.

The project focuses on building a production-style machine learning pipeline and deploying the trained model as an API.

Main components of the system include:

- Data ingestion and storage using MySQL
- Data preprocessing and feature engineering using Pandas and Scikit-learn
- Model training using ensemble models such as Random Forest and XGBoost
- Experiment tracking using MLflow
- REST API development using FastAPI
- Docker containerization
- Deployment on AWS EC2
- Data visualization using Power BI dashboards

Repository:
github.com/Mukul9112003/project1_Salary_Intelligence_Platform



### 2. Hybrid Movie Recommendation System

This project implements a hybrid recommendation system that combines collaborative filtering and content-based filtering techniques.

The goal of the system is to generate personalized movie recommendations by combining user behavior data with movie metadata.

The recommendation pipeline includes:

1. Candidate generation using collaborative filtering
2. Content-based filtering using TF-IDF vectorization
3. Meta-ranking using machine learning models such as Logistic Regression and Support Vector Machine
4. Evaluation using ranking metrics including Precision@K, Recall@K, and NDCG@K

The system provides real-time recommendations through a FastAPI service and includes a Streamlit interface for user interaction.

Repository:
github.com/Mukul9112003/Project2_Hybrid_Movie_Recommendation_System



### 3. Telecom Customer Churn Prediction System

The objective of this project is to predict which telecom customers are likely to leave a service provider.

Customer churn prediction is important for telecom companies because retaining existing customers is often more cost-effective than acquiring new ones.

The machine learning workflow includes:

- Data preprocessing and cleaning
- Handling class imbalance using SMOTE
- Model training using Random Forest and XGBoost
- Experiment tracking using MLflow
- Model explainability using SHAP
- Deployment of an interactive prediction application using Streamlit
- Containerization using Docker

Repository:
github.com/Mukul9112003/project3_Telecom_Customer_Churn_Prediction_System
